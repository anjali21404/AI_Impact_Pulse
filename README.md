# AI_Impact_Pulse
Interactive Power BI dashboard analyzing AI adoption trends, workforce insights, and regional patterns using survey data.
# Overview
AI Impact Pulse is a data visualization project that analyzes how artificial intelligence is transforming the IT industry across roles, regions, and experience levels. The project presents insights using interactive Power BI dashboards built on real-world survey data.

# Purpose
The goal of this project is to understand AI adoption trends, workforce sentiment, and skill transformation in the software industry using data-driven analysis and visualization.

# Features
- Interactive Power BI dashboards with role, region, and experience-based filters
- Analysis of AI usage frequency, productivity impact, and professional sentiment
- Visualization of declining skills and emerging AI-driven skill areas
- KPI-based insights for workforce and industry trends

# Tech Stack
- Power BI
- Google BigQuery
- Python
- pandas, NumPy
- Microsoft Excel

# Workflow
1. Survey data collection using Google Forms
2. Data cleaning and preprocessing in Excel and Python
3. ETL pipeline implementation using BigQuery
4. Interactive dashboard creation in Power BI

# Impact
AI Impact Pulse provides clear, data-driven insights into how AI is reshaping modern software roles, helping students, professionals, and organizations understand evolving skill demands and workforce trends.
